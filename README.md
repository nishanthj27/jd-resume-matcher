# JD-Resume Matcher

An AI-powered tool that analyzes how well your resume matches a job description, providing detailed insights and improvement suggestions.

## Features

- 📊 **Match Percentage**: Get an overall compatibility score
- 🎯 **Skills Analysis**: See which skills match and which are missing
- 📈 **Experience Matching**: Understand if your experience level aligns
- 🔍 **Keyword Analysis**: Discover important keywords you're missing
- 💡 **Smart Suggestions**: Get actionable tips to improve your resume

## Demo

[Live Demo](https://jd-resume-matcher.vercel.app/)

## Technology Stack

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Python (Serverless)
- **NLP**: Custom algorithms for text analysis
- **Hosting**: Vercel (Free tier)

## How It Works

1. **Text Input Only**: Paste your resume and job description (no file uploads)
2. **Instant Analysis**: Get results in seconds
3. **Detailed Breakdown**: See exactly what matches and what doesn't
4. **Actionable Advice**: Receive specific suggestions for improvement
